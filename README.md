# smart-campus-network-CCNA-
Smart Campus Network Monitoring – A Cisco Packet Tracer–based network simulation designed using CCNA 200-301 principles, demonstrating VLAN segmentation, inter-VLAN routing, DHCP configuration, and structured IP addressing for a campus environment
